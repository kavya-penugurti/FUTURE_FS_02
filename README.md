# Mini CRM

A simple CRM application to manage leads and customer information.

## Features
- Add new leads
- View lead list
- Update lead status
- Delete leads

## Technologies Used
- Node.js
- JavaScript
- HTML
- CSS

## How to Run
1. Install dependencies:
   npm install

2. Start the server:
   npm start

3. Open the application in your browser.
